DecHW: Second-Order Decentralized Federated Learning

A PyTorch-based decentralized training framework with Hessian-based aggregation for heterogeneous peer-to-peer federated learning.

Overview

DecHW addresses decentralized federated learning where clients collaborate over peer-to-peer networks without a central server. The method uses second-order information to weight model parameters during neighborhood aggregation.

Key Contributions

  • Built a PyTorch-based decentralized FL training environment.
  • Implemented peer-to-peer model exchange over graph-based communication topologies.
  • Developed DecHW, a Hessian-based aggregation method for parameter-level weighting.
  • Validated against standard baselines including FedAvg-style aggregation.

Tech Stack

Python, PyTorch, Federated Learning, Distributed Training, Graph Networks, Non-IID Data.
