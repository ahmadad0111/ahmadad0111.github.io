Adaptive Human-AI Coordination via Physiological Feedback

A real-time Overcooked-based human-AI coordination system where physiological state prediction models infer human mental state and reinforcement learning agents adapt skills accordingly.

Overview

This project integrates reinforcement learning agents with physiological feedback from EEG, ECG, and eye-tracking streams. The goal is to enable AI agents to adapt their behavior based on the human collaborator's inferred cognitive or emotional state and provide interpretable action explanations.

Problem

Human-AI coordination in dynamic environments requires AI agents that can respond not only to the task state, but also to the human collaborator's mental state. Standard RL agents often optimize task reward without accounting for cognitive load, stress, or emotional changes.

What I Built

  • Integrated Overcooked-AI with adaptive reinforcement learning agents.
  • Connected physiological state prediction models to the agent decision pipeline.
  • Used EEG, ECG, and eye-tracking streams to infer human mental state in real time.
  • Enabled RL agents to adapt skill selection based on predicted human state.
  • Added interpretable action explanations to communicate why the agent selected a behavior.
  • Supported real-time operation using LSL, Docker, and FastAPI-based APIs.

Tech Stack

Reinforcement Learning, PPO, Overcooked-AI, PyTorch, EEG, ECG, Eye-tracking, LSL, Docker, FastAPI, GCP.

Impact

The system demonstrates how implicit physiological feedback can be used to improve adaptive human-AI collaboration by connecting real-time human state prediction with RL-based skill adaptation and explainability.
