Human-AI Systems · Reinforcement Learning · Physiological Feedback

Adaptive Human-AI Coordination via Physiological Feedback

A real-time Overcooked-based system where physiological state prediction models infer human state and RL agents adapt skills accordingly.

Overview

This project connects physiological state prediction models with adaptive reinforcement learning agents. EEG, ECG, and eye-tracking streams are used to infer human mental state, which is then used by agents to adapt skill selection and provide interpretable action explanations.

Key Contributions

  • Integrated Overcooked-AI with adaptive reinforcement learning agents.
  • Connected physiological state prediction models to the agent decision pipeline.
  • Enabled agents to adapt skill selection based on human state feedback.
  • Added interpretable explanations for agent actions.

Tech Stack

Reinforcement Learning, PPO, PyTorch, EEG, ECG, Eye-tracking, LSL, Docker, FastAPI, GCP.
