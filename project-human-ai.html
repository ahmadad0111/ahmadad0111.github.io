Real-Time Human–AI Coordination via Physiological Feedback

An end-to-end system that integrates physiological state prediction with adaptive reinforcement learning agents, enabling human-aware decision-making in real time.

Overview

This project develops an end-to-end human–AI coordination system in a multi-agent environment (Overcooked), where physiological signals (EEG, ECG, and eye-tracking) are used to infer human cognitive and emotional state in real time. These signals are streamed and processed through a low-latency pipeline and integrated into the decision loop of reinforcement learning agents.

The agents leverage this feedback to adapt skill selection dynamically, enabling more effective coordination with human partners while also providing interpretable explanations of their actions.

System Design

  • Real-time multimodal data streaming and synchronization using Lab Streaming Layer (LSL).
  • Physiological state prediction models for estimating cognitive and emotional state.
  • Reinforcement learning agents (PPO-based) operating in a multi-agent Overcooked environment.
  • Integration of human state into the agent policy for adaptive decision-making.
  • Containerized inference and API-based deployment using Docker and FastAPI.

Key Contributions

  • Designed an end-to-end pipeline connecting physiological sensing to RL-based decision-making.
  • Enabled human-aware adaptation through real-time cognitive and emotional state feedback.
  • Developed interpretable agent behavior via explanation of actions in context.
  • Demonstrated adaptive coordination in a multi-agent environment.

Tech Stack

Reinforcement Learning (PPO), PyTorch, EEG/ECG Processing, Eye-tracking, Real-time Data Streaming (LSL), Docker, FastAPI, Google Cloud Platform (GCP).
