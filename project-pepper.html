Pepper Robot Conversational AI System

A real-time conversational AI system integrating SoftBank Pepper with GPT-based APIs for interactive campus assistance and demonstrations.

Overview

This project connects a humanoid robot with cloud-based language model inference to support natural conversational interaction in public-facing demonstrations.

Key Contributions

  • Integrated SoftBank Pepper robot with GPT-based conversational APIs.
  • Designed an API-driven architecture connecting speech input, cloud inference, and robotic responses.
  • Developed middleware for low-latency communication between robot hardware and AI services.

Tech Stack

Python, GPT APIs, Robotics Middleware, Speech Interaction, API Integration.
