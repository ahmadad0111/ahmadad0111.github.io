Real-Time Conversational AI for Pepper Robot

An end-to-end conversational AI system integrating a humanoid robot with cloud-based language models for real-time, interactive human–robot engagement.

Overview

This project develops a real-time conversational AI system for the SoftBank Pepper robot, enabling natural language interaction in public-facing environments such as campus demonstrations and outreach events.

The system connects speech input, cloud-based language model inference, and robotic actuation through an API-driven architecture, enabling low-latency, context-aware dialogue between users and the robot.

System Design

  • Speech-to-text processing for real-time user input.
  • Cloud-based language model inference using GPT-style APIs.
  • Middleware layer for communication between robot hardware and AI services.
  • Text-to-speech and motion integration for synchronized verbal and physical responses.
  • API-based architecture enabling modular and scalable deployment.

Key Contributions

  • Designed an end-to-end pipeline for real-time human–robot interaction.
  • Integrated Pepper robot with cloud-based conversational AI services.
  • Developed low-latency middleware for seamless communication across system components.
  • Enabled interactive demonstrations for large-scale public engagement.

Tech Stack

Python, GPT APIs, Speech Processing, Robotics Middleware (NAOqi), API Integration, Real-Time Systems.
