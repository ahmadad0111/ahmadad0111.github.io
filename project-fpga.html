Pipelined MIPS Processor & Vision Accelerators on FPGA

An end-to-end hardware–software system combining a custom pipelined MIPS processor, FPGA-based vision accelerators, and a Java-based assembler for instruction translation.

Overview

This project focuses on computer architecture and FPGA-based acceleration. I designed and implemented a pipelined MIPS processor in Verilog HDL, developed optimized hardware accelerators for computer vision workloads, and built supporting software tooling to translate assembly instructions into machine code.

The complete hardware–software pipeline was validated on Xilinx Virtex-5, with design trade-offs analyzed across latency, area, and throughput.

System Design

  • Pipelined MIPS processor architecture implemented in Verilog HDL.
  • Low-area and high-performance FPGA accelerator designs for computer vision workloads.
  • Java-based assembler for translating assembly instructions into machine code.
  • Hardware validation and testing on Xilinx Virtex-5 FPGA.
  • Performance analysis across latency, area, and throughput trade-offs.

Key Contributions

  • Designed a complete processor pipeline with hardware-level instruction execution.
  • Developed custom accelerator variants optimized for area and performance constraints.
  • Built software tooling to support end-to-end hardware–software integration.
  • Validated the system on FPGA hardware rather than only simulation.

Tech Stack

Verilog HDL, Xilinx Virtex-5, Java, FPGA Design, Computer Architecture, Digital Design, Hardware–Software Co-design.
