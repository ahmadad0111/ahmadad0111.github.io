FPGA MIPS Processor & Vision Accelerators

A pipelined MIPS processor and optimized computer vision accelerators validated on Xilinx Virtex-5.

Overview

This project combines digital hardware design, FPGA validation, and supporting software tooling for a custom MIPS processor and computer vision accelerators.

Key Contributions

  • Designed and implemented a pipelined MIPS processor using Verilog HDL.
  • Validated the design on Xilinx Virtex-5 FPGA hardware.
  • Optimized low-area and high-performance vision accelerators.
  • Developed a Java-based assembler to translate assembly instructions into machine code.

Tech Stack

Verilog HDL, Xilinx Virtex-5, Java, FPGA, Computer Architecture, Digital Design.
