# Adnan Ahmad Portfolio Website

This is a simple GitHub Pages portfolio website.

## Files

- `index.html` — homepage
- `style.css` — styling
- `project-human-ai.html` — adaptive human-AI coordination project
- `project-dechw.html` — DecHW project
- `project-pepper.html` — Pepper robot project
- `project-fpga.html` — FPGA project
- `publications.html` — publications page
- `assets/` — place your CV PDF and images here

## What to edit first

1. Replace placeholder GitHub links:
   - Search for `https://github.com/yourrepo`
2. Replace Google Scholar link:
   - Search for `https://scholar.google.com/`
3. Add your CV PDF:
   - Put it in `assets/Adnan_Ahmad_CV.pdf`
4. Check email, LinkedIn, and GitHub links in `index.html`.

## Deploy on GitHub Pages

1. Create a GitHub repository named:

   `yourusername.github.io`

2. Upload all files to the repository.

3. Go to:

   Settings → Pages → Deploy from branch → main → root

4. Your website will be available at:

   `https://yourusername.github.io`
